        • org.mini2Dx.core.graphics.Sprite getSprite(int id, int size)

          Get a sprite from one of the 4 banks. If the id is outside of the index range of a bank, the error screen will be invoked with the message in the catch. If a localized message exists, it will append it. The first sprite in default bank is returned to prevent followup errors in the EngineScreen render method.

          Parameters:
          id - The id of the sprite to return
          size - Which bank size to index
          Returns:
          The sprite given the ID and size
